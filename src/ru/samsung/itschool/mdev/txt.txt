класс - сущность,объединение свойств, финкций и значений, что упрощает
 код визуально и технически.
объект - экземпляр класса, что выполняет некии связывающие функции.

Инкапсуляция - создание класса, выполняющего множество дествий, получая данные
 из простого интерфейса

Абстракция - это формирование наипростейшего интерфеса, отделение значимой инфо

Наследование - сохранение свойств наследуемого класса, для уменьшения длины кода, используется когда в классах есть значимая общая часть.

Полиморфизм - сохранение общих свойств под одной переменной,
чтобы закрепить несколько алгоритмов.

Модификаторы доступа:

public - члены класса доступны всем
Private - только внутри класса
default - виден только внутри пакета (по-умолчанию)
Protected - доступны внутри пакета и в наследуемых пакетах
